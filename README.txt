# eNDR Heart Disease Risk Model

This repository contains a neural model enhanced with Extended Nomological Deductive Reasoning (eNDR) for cardiovascular disease risk prediction.

## Structure

- `data/`: Raw dataset CSV (download and place your heart disease dataset here)
- `models/`: Model code with BaseModel, LawModules, and combined eNDRModel
- `utils/`: Preprocessing, metrics, and explanation helpers
- `lightning/`: PyTorch Lightning module wrapper
- `train.py`: Training script
- `eval.py`: Evaluation and reporting script
